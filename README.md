# myStockMarket

Single-user US-equities command center + learning hub. A calm, editorial web app and installable
mobile PWA whose after-close cloud pipeline writes a verified evening briefing — and which never
predicts direction.

**Status:** planning complete; build not started. Everything under `app/` and `pipeline/` is
built phase-by-phase (P0–P6) per the plan below.

## The documents
| File | Role |
|---|---|
| `DEVELOPMENT-PLAN.md` | The executable build contract (markdown working copy of the PDF) |
| `docs/Development-Plan.pdf` | Same content, typeset (35 pp) |
| `docs/Research-Report.pdf` | Evidence + product design — Parts 8/9 are the top-ranked authority |
| `docs/Build-Blueprint.pdf` | Architecture, stack, data plan, P0–P6 roadmap, risks |
| `docs/src/*.html` | Grep-able sources of all three PDFs (rr-\*, bp-\*, dp-\*) |
| `CLAUDE.md` | Standing constitution for the executor |
| `DECISIONS.md` | Append-only decision log + the user's veto channel |
| `PROGRESS.md` | Resumable checkpoint — read this to orient any session |

Authority when sources disagree: Research Report Part 8/9 → Blueprint → Development Plan →
DECISIONS.md → judgment (plan §1.2).
